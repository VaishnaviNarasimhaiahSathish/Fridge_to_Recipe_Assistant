# Fridge-to-Recipe Assistant

This project builds a VLM-focused fridge-to-recipe assistant.

The main goal is to identify ingredients from fridge images using Vision-Language Models and recommend suitable recipes based on the detected ingredients.

YOLO/object detection is used only as a comparison baseline, not as the main pipeline.
